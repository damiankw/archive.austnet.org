Would you like your website to have a java chat applet like this to connect to your channel on Austnet?

We now have a Java Chat Applet that you can download and put on your website. All instructions are provided in a zip file as well as a standard webpage if you do not want to configure the chat yourself.

Applet Download
Applet help



Try out the Austnet Java IRC Client (JAustIRC) here to see if it is what you want for your website. This demo will join you to #AustNetWebsite. From there you can join other channels.

You will need to grant permission when the pop up window appears from Netscape and IE. Enter your desired nickname (no spaces) and User ID (something you make up - can be the same as your nickname).

The JAustIRC applet only supports Netscape 4.0, IE 4.0 or above.
Sorry, your browser does not support Java. Please download the Java Virtual Machine from here: http://java.sun.com/products/plugin/autodl/jinstall-1_4-windows-i586.cab#Version=1,4,0,0

The chat above uses the default colours and options. Many of these can be customised for your own website.
If you would like this JAustIRC chat on your website, download all the files from our Applet download page, and use our help page to set it up. Happy chatting!
