DCC

Why can't I send files or DCC chat?

The usual reason is that you're using mIRC and you have an incorrect IP address specified. Go to the Setup menu and open the Local tab. Change all the options so that the Local Host box has a check mark inside it (the IP Address box will grey out) inside the On connect, always get: box. Then make sure the Lookup method is set to Server. AustNet's IRC server software sends your own IP address to mIRC so it can configure DCC for you.

Other reasons why DCC may not work include:

  • You're on a network that's not set up to allow DCC.
  • You are using a SOCKS Firewall.
  • You use SLIRP or another PPP/SLIP-over-shell program which doesn't allow DCC.
There's not much you can do about this.
