What software do I need?
To make the best use of IRC you will need a program called an IRC client. You use the client to connect to a server; there are many different clients but they can all be used on the same servers. The most common Windows client is mIRC. You can download an IRC client from our software pages.
You should pick an IRC client that will suit your operating system:
Unix: client software for UNIX based systems.
Windows: client software that suits Microsoft Windows 95 / 98 / ME / XP / NT / 2000.
Mac: client software that can be used for Apple Macintosh computers.

What is a "bot"?
A bot is a program that looks like a user and connects to IRC in the same method as a user. Bots are not real people. The bot is there to do certain functions such as run games or server 'virtual drinks', for example.

Can I have a bot on AustNet?
For a start, you need to know AustNet rules regarding bots. A bot are allowed on AustNet provided it
is not a flood/clone bot;
it is not a takeover bot;
it does not duplicate what AustNet services does (i.e. ChanOP)
it does something other than sit in an unused channel.

Where can I get a bot?
You can get an eggdrop (an Open Source IRC bot) from http://www.eggheads.org/
Other bots can be sourced in various scripting languages. For more help try typing /list bot on AustNet.

What are scripts?
A script is a collection of commands, written in a scripting language, that customises your IRC client for your use.

Where can I get a good script for mIRC?
A new AustScript site designed specifically for AustNet is due to be released in April 2003. Until its ready, you can ask in #mirc for safe scripts.
Be aware that not all scripts are safe, and some may contain "backdoors" that allow other users access to your computer, or may contain viruses.


For more help on software, you can ask in #help, or contact help@austnet.org.


The coder of mIRC, the most popular IRC client, included many little "tricks" in the software. Try these:
Click on Khaled's nose in the About box to "squish" it.

Type "arnie" anywhere into the about box for the faithful pet.

Click on the mIRC logo to see the older, more tasteful version.

Tick the checkbox at the bottom of the About box to avoid seeing it on startup.

Right click anywhere in the About box, and the dot on the "i" bounces.

right click on the 'about' icon and little messages appear.
