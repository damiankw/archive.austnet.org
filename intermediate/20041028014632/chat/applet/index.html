Would you like your website to have a java chat applet to connect to your channel on AustNet? Are you having a special event in your channel and would like web access?
You can now download a free chat applet specially designed for AustNet use, for use on your website!

Simply follow these easy steps:

The zip file contains everything you need to install JAustIRC on your website.


Next you need to unzip the file. You will need winzip or some other unzipping utility. In the zip file you will find the following files:

readme.txt - a file with this information in it.
sjavaIRC.jar - the actual java archive file you will need to use
install.html - instructions for installing the chat applet on your website.
configure.html - instructions on how to change the options on the chat applet - colours, quit message etc.
jaustirc.html - a html template that you can use readymade on your site.


Once you have unzipped the file, you can either use the jaustirc.html template for your site 'as is', or you can configure the client to match your website colours, have your own customised quit message and more.
Full instructions for customisation are given in the file configure.html from the zip file you downloaded, or can be found on our help page.


Once you have configured the client, you can upload the html file (either jaustirc.html or your own custom html file) and the sjavaIRC.jar files to your website.
Important: the sjavaIRC.jar file must be in the same folder/directory as the html file, or the java chat will not function.


Use your new java chat to connect to your channel on AustNet and chat!

If you have problems configuring the applet, please use our demo page to connect to #AustNetWebsite, where someone will most probably be able to assist you.
If you have any problems with the applet, or have a bug to report, email Sophie as much information about the problem as possible. Please note, that this is only for help with installing or using the applet, and not for help with any of AustNet’s services.
