The IRCG (Internet Relay Chat Gateway) was designed to link all IRC networks around the world via a message bank gateway. This allows users to talk to each other regardless of what network they are connected to. This website explains, in detail, how this is achieved along with information in linking your network and talking through the inter-network gateway.


Unlike any other gateway, the IRCG is built with client flood protection in mind. It prevents thrashes of messages transfered over the inter-irc networks amongst users, and thus cannot be used as a tool for 'network wars'. Featuring the ability to disallow clients and userhosts from using the gateway, the IRCG is able to provide a safe method of communication between networks.
