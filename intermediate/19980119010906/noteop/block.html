AustNet's notebook service allows you to block a user from sending you a note. This is a useful command if someone is abusing you on AustNet.
  • Block Commands
    • This command will let you block out a user from sending you a note,
    • SYNTAX: /msg NoteOP block [list|add|del] [block mask]
    • EXAMPLE: /msg NoteOP block list
    • EXAMPLE: /msg NoteOP block add *!*@aussie.net
    • EXAMPLE: /msg NoteOP block add webmaster
    • EXAMPLE: /msg NoteOP block del Rogery
    • EXAMPLE: /msg NoteOP block add *@*wantree.com.au
    • EXAMPLE: /msg NoteOP block add #beginner*
