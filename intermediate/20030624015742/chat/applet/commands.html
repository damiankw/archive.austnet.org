Service Commands
The most common service commands for ChanOP, LoveOP, NoteOP and NickOP are available via the menu bar on the main window. If you don’t want to use these, you can use the /msg command in the input field.

ChanOP commands - Op, deop, voice, devoice, kick
NickOP commands - identify, kill nick, register nick
LoveOP commands - admire, apology, birthday, block, marriage, candy, chocolate, cookie, flowers, smooch, hug, love, love note, present, vote.
NoteOP commands - Read, send, list and delete notes.

Mode commands
You can perform user and channel MODE commands through the user and channel menus. The command line MODE command (/mode) is not yet supported. Right clicking on a channel window will allow you to change modes via a popup menu. Right clicking on a nickname in the nick list will allow you to change user modes via ChanOP.

/msg – this allows you to send a message to a user
Syntax:
/msg <username> <message>

/query – this opens a private chat window to a user
Syntax:
/query <username>
/q <username>

/join – this joins a channel with its own channel window
Syntax:
/join <channel>
/j <channel>

/part – leaves a channel
Syntax:
/part <channel>
/p <channel>

/nick – this will change your nickname
Syntax:
/nick <new nickname>
/n <new nickname>

/topic – views a topic for a channel
Syntax:
/topic <channel>
/t <channel>

/whois – show the details of a user
Syntax:
/whois <username>
/w <username>

/me – send an action message to a channel or user
Syntax:
/me <message>

More help
If you have any problems with the applet, or have a bug to report, email sophie@austnet.org with as much information about the problem as possible. Please note, that this is only for help with installing or using the applet, and not for help with any of AustNet’s services. You can also join #austnetwebsite on the AustNet IRC Network, and we will attempt to help you.
