New website live

By ryan 0 Comments
Categories: Administration

As of February 14, we have installed a new Wordpress for the AustNet.org web server.  This provides blog-based website functionality, and various AustNet members are able to post to our website with thoughts, suggestions, latest news and information pertaining to the network and IRC here.  Keep an eye on the site as we begin to upload more content.  Any enquiries? Simply email admin@austnet.org

Users wanting to register and contribute to the website are able to do via the Registration Page
