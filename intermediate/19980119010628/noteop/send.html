AustNet's notebook service allows you to send notes to registered nicknames and registered channels on the AustNet network.
  • Send Commands
    • This will let you send a note to a nickname, The user getting the note doesn't have to be online at the time to recieve the note(s).
    • SYNTAX: /msg NoteOP send [nicks|chan] [Level] [message]
    • EXAMPLE: /msg NoteOP send redtrio Hi Redtrio, NoteOP me
    • EXAMPLE: /msg NoteOP send redtrio,Seecs This message is being sent to redtrio and Seecs.

    • This command will let you send notes to everyone who has a certain amount of access on a channel.
    • SYNTAX: /msg NoteOP send [#channel] [level] [message]
    • EXAMPLE: /msg NoteOP send #help 125 Services are running smoothly
    • That example will send a message to everyone on #help with an access level of 125.

    • NOTE: To send to multiple users use a comma between nicknames, eg. redtrio,Seecs
