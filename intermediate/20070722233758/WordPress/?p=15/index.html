Registration required

post info

By ryan
Categories: Services

Certain channels have now utilised the new channel mode +R — this channel mode requires that users be authenticated with the AustNet.org IRC services before being able to join the channel.

you will need to /msg nickop@austnet.org identify [password] before being able to join any channel that has +R set. if you have not already registered your nickname, simply /msg nickop help register when you’re next online. if you need further assistance with this, feel free to /join #help or /join #asd
