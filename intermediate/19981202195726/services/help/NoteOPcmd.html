Service Information
NoteOP allows users to send short memos to each other. If the user to is not online, they will be notified of new notes when they return.

Service Help Library
For further information on any of these commands, use specific help inquiries, /msg NoteOP HELP <command>.

LIST
Syntax: /msg NoteOP list [new]
Access:you must be identified as the owner.
Example:/msg NoteOP list (This will list all your notes new or old)
Example:/msg NoteOP list new (This will list all your new notes)

SEND
Syntax: /msg NoteOP send <nick(s)|#chan> [level]
Access:you must be identified as the owner.
Example:/msg NoteOP send Woger Hay, where ya been? (This will send a note to Woger saying 'Hay, Where ya been?')
Example:/msg NoteOP send Woger,RogerY Hay, where ya been? (This will send a note to both Woger and RogerY saying 'Hay, Where ya been?.')
Example:/msg NoteOP send #help I need some help here (This will send a note to everyone on #help's database)
Example:/msg NoteOP send #help >100 Hiya guys with more than 100 access =o) (This will send a note to those in the #help database with more than 100 access)
Example:/msg NoteOP send #help <90 Hiya guys with less than 90 access =P (This will send a note to those in the #help database with less than 90 access)

READ
Syntax: /msg NoteOP read <number>
Access:you must be identified as the owner.
Example:/msg NoteOP read 12 (This will display note # 12 to you)

DEL
Syntax: /msg NoteOP del <number>
Access:you must be identified as the owner.
Example:/msg NoteOP del 12 (This will mark note #12 as (D)eleted)

PURGE
Syntax: /msg NoteOP purge
Access:you must be identified as the owner.
Example:/msg NoteOP purge (This will remove all notes marked (D)eleted from your notebox)

BLOCK
Syntax: /msg NoteOP block <list|add|del> <block mask>
Access:you must be identified as the owner.
Example:/msg NoteOP block list (This will list all your current blocks)
Example:/msg NoteOP block add redtrio (This will block all redtrio's notes from being sent to me)
Example:/msg NoteOP block add #help (This will block all #help's notes from being sent to me)
Example:/msg NoteOP block add *!*@*.aol.com (This will block all notes from *!*@*.aol.com being sent to me)
Example:/msg NoteOP block del #help (This will remove my block of #help)
Example:/msg NoteOP block del redtrio (This will remove my block of redtrio)
Note:wildcards can be used e.g. #*help, red*trio
