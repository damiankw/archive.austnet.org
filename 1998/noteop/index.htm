AustNet's Notebook services allows a user to send a note to a registered user or an entire channel. Even if the user is not online at the time the note is still sent, as soon as that user comes online, they will recieve the note. As extra security, in order to recieve your notes from NoteOP, the user must identify with NickOP first, preventing any unauthroized reading of notes.
  • NoteOP Commands
    • LIST - lists all current read, unread, and new notes.
    • SEND - sends a note to a registered nickname or registered channel.
    • READ - reads a specified note.
    • DEL - deletes notes which have been read.
    • PURGE - purges out all deleted notes.
    • BLOCK - blocks a user's access to send you a note.
