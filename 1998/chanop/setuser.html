-ChanOP- *** Command Help ***
-ChanOP- Use setuser to modify access and flags of channel members.
-ChanOP- These settings affect the status and operation of users
-ChanOP- ability to use the service. 
-ChanOP-  
-ChanOP- For more information on how to use the below settings, type
-ChanOP- /msg ChanOP help setuser <setting>
-ChanOP-  
-ChanOP- Setting    Description                           
-ChanOP- level      modify access level
-ChanOP- aop        auto-op setting on join
-ChanOP- aov        auto-voice setting on join. Only if you dont have ops.
-ChanOP- protect    protect the user from kick,deops
-ChanOP- suspend    will suspend a users access for the specified time
