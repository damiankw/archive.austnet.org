-ChanOP- *** Command Help ***
-ChanOP- Telldel is used when you (the owner) wants ChanOP to send you a
-ChanOP- note each time someone in your database has been deleted by -ChanOP-someone else.  If you wish to know who modified records by adduser  -ChanOP-or setuser then use...
-ChanOP- /msg ChanOP access #channel -lm    (lm means last modified).
-ChanOP- 
-ChanOP- *** Command Syntax ***
-ChanOP-    Syntax: set <#chan> telldel <on/off>
-ChanOP-    Access: Requires (200) channel access.
-ChanOP-  Examples:  /msg ChanOP set #chat telldel on
-ChanOP-                    (Owner gets a note from ChanOP when user record deleted)
-ChanOP-  See Also: SET, INFO
