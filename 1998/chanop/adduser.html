-ChanOP- *** Command Help *** 
-ChanOP- The addition of users to a channel database is a simple 
-ChanOP- process. As a pre-requisite, the user you wish to add to 
-ChanOP- the database must have their nickname registered. To add 
-ChanOP- a user to your channel, you must also deligate a user 
-ChanOP- level.  For further information regarding access levels, 
-ChanOP- simply /msg ChanOP help levels. If a user level is not 
-ChanOP- specified, a default of (100) access is issued. 
-ChanOP-   
-ChanOP- *** Command Syntax *** 
-ChanOP-    Syntax: adduser <channel> <nick> [level] 
-ChanOP-    Access: Requires (150) channel access. 
-ChanOP-  Examples:  /msg ChanOP adduser #AustOP Cinder 150 
-ChanOP-                  (Addition of Cinder to #AustOP at 150 access) 
-ChanOP-                      /msg ChanOP adduser #AustOP Poseidon 
-ChanOP-                  (Addition of Poseidon with 100 access) 
-ChanOP-  See Also: DELUSER, SETUSER, ACCESS
