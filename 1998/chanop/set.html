-ChanOP- *** Command Help ***
-ChanOP- Using the set command allows channel managers to
-ChanOP- configure their channel's various settings. These
-ChanOP- settings affect status and operation of a channel.
-ChanOP-  
-ChanOP- For more information on how to use the below settings
-ChanOP- /msg ChanOP help set <setting>
-ChanOP-  
-ChanOP- Setting    Description                           
-ChanOP- restrict        require certain access to join channel
-ChanOP- keeptopic   enforce the topic to maintain the same
-ChanOP- url               display the url to users who join
-ChanOP- mode           set a mode on the channel
-ChanOP- oprestrict   only people on chan database allowed ops
-ChanOP- mustid         Database must identify for autoop/protect
-ChanOP- telldel         Owner gets note when someone uses deluser
-ChanOP- nonote        Level 200 only can send chan notes
