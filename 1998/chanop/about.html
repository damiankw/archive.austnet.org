-ChanOP- *** Command Help ***
-ChanOP- This command reports user access levels on a specificied
-ChanOP- channel. You can review the entire channel database or
-ChanOP- review individual user records.  For further information
-ChanOP- relating to user levels, /msg ChanOP help levels.
-ChanOP- Criteria includes [You may use any combination in any order]:
-ChanOP-  -min level  Users greater than the specified level
-ChanOP-  -max level  Users smaller than the specified level
-ChanOP-  -lm         Who last modified (adduser/setuser) record
-ChanOP-  -ls         When were the users last seen
-ChanOP-  *nick*      Users matching wildcard *nick*
-ChanOP-  
-ChanOP- *** Command Syntax ***
-ChanOP-    Syntax: access <channel> <criteria>
-ChanOP-    Access: No channel access is required.
-ChanOP-  Examples:  /msg ChanOP access #AustOP redtrio
-ChanOP-                     (Display redtrio's access to #AustOP)
-ChanOP-                       /msg ChanOP access #AustOP * -lm
-ChanOP-                     (Display all records for #AustOP and last modified)
-ChanOP-                        /msg ChanOP access #AustOP * -min 75
-ChanOP-                     (Display records with level 75 or higher)
-ChanOP-                       /msg ChanOP access #AustOP -max 130 -min 2 *rog*
-ChanOP-                     (Display records with level less than 130 but greater
