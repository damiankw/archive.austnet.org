ChanOp was one of AustNet's Original Services. Since it was first coded, it has been immensely devolped into a true 'Channel Operator'. With the ChanOp service, users have the ability to register and control a channel with their own channel ops, their own specific channel specifications, and their own idea of what their 'ideal' channel should be like. The following are commands available to users for ChanOp. 
  • ChanOP Level 0 Channel Commands
    • ABOUT - displays information about ChanOp
    • COMMANDS - allows a user to see the commands that he/she has access to in a channel. 
    • HELP - displays all of chanop's commands. 
    • INFO - displays information pertaining to a channel. 
    • LIST - lists all the channels registered to a user. 
    • LISTBAN - displays all the bans for a channel and information about each. 
    • REGISTER - allows a user to register a channel as his/her 'own'. 
  • ChanOP Level 25 Channel Commands
    • ACCESS - tells you your access level in a channel. 
    • INVITE - allows you to invite people to a channel. 
  • ChanOP Level 50 Channel Commands
    • KICK - allows a user to kick another user from a channel. 
    • TOPIC - allows a user to set the topic for a channel. 
  • ChanOP Level 75 Channel Commands
    • BAN - bans a user from a channel for a 'set' amount of time. 
    • UNBAN - unbans a banned user from achannel. 
  • ChanOP Level 100 Channel Commands
    • CLEAN - removes all the modes in a channel. 
    • DEOP - deops an op in a channel. 
    • OP - ops a user in a channel. 
  • ChanOP Level 150 Channel Commands
    • ADDUSER - adds an access level to a user in a channel. 
    • DELUSER - removes a user's access to a channel. 
    • SETUER - modifies a user's level in a channel. 
    • JOIN - tells chanop to join a specified channel. 
    • PART - tells chanop to part a channel. (NOTE all commands will still function with chanop not in the channel) 
    ChanOP Level 200 Channel Commands
    • DROP - drops or deregisters a channel. 
    • MDEOP - takes away ops from everyone with a lower access level in the channel. 
    • SET - sets a restrict, keeptopic, url, mode, and op restrect in a channel. 
NOTE 
  • Each increasing level has access to all commands below that level. 
  • There can be more then 1 level 200 for a channel, come to #help for more assistance.
