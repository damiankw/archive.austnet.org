IRC Client
You can connect to the Austnet using the same client as you would for most other IRC networks, you just connect to an AustNet IRC Server. All servers on the AustNet IRC Network accept client connections on ports 6667-6669. If you find port 6667 is congested, feel free to try ports 6668 and 6669.

  • For a comprehensive list of AustNet Servers, click here.

AustNet has round robins set up for users so you can connect to a random IRC server within a specific regional area. The following are areas covered by AustNet and servers you can connect to:
  • irc.austnet.org - International Round Robin
  • us.austnet.org - United States of America
  • au.austnet.org - Australia
  • ca.austnet.org - Canada
  • sg.austnet.org - Singapore
  • nz.austnet.org - New Zeland
