Software

What software do I need for IRC?

To make the best use of IRC you will need a program called an IRC client. You use the client to connect to a server; there are many different clients but they can all be used on the same servers. The most common Windows client is mIRC. You can download an IRC client from our software mirror.

Can I run "bot" programs? Where do I get one?

Each user is allowed one connection to AustNet. That means if you can't run a bot from your own computer and load an IRC session as well, therefore we recommend you run it on a shell server. Bots may only be run on the following servers: 
aussie.nsw.au.austnet.org
iinet.wa.au.austnet.org
firstlink.nsw.au.austnet.org
ozemail.nsw.au.austnet.org
powerup.qld.au.austnet.org
wantree.wa.au.austnet.org
shadow.il.us.austnet.org
webmaster.ca.us.austnet.org
The best place to get a bot is sodre.net.

What are some of the tricks in mIRC?

  • Click on Khaled's nose in the About box to squish it.
  • Type "arnie" anywhere into the about box for the faithful pet.
  • Right click in the about box for the bouncing dot.
  • Click on the mIRC logo to see the older, more tasteful version.
  • Tick the checkbox at the bottom of the About box to avoid seeing it on startup.
  • Right-click toolbar buttons repeatedly to see weirdo messages.
