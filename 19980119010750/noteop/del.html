AustNet's notebook service allows you to delete notes sent to you from registered nicknames, Use this command wisely.
  • DEL Commands
    • A user can use the DEL command to delete specific messages or all messages in his/her notebox.
    • SYNTAX: /msg NoteOP del [numbers|all]
    • EXAMPLE: /msg NoteOP del 2
    • The example deleted note #2.
    • EXAMPLE: /msg NoteOP del 2,5,6
    • The example deleted notes 2,5,6
    • EXAMPLE: /msg NoteOP del all
    • The example deleted all notes in the users notebox
