There are various methods of contacting AustNet based upon who the audience of your request should be. Simply click on the any of the following mail links to contact AustNet:

  • Client Support Mailing List
    For assistance with any AustNet related problems or issues, you can mail a help mailbox and the network will reply as soon as possible. Topics normally include connection details, service assistance and general network information. It is estimated your post will be replied within one hour or less during normal business hours.

    • Individual help requests should be simply mailed to help@austnet.org
    • Permanent subscription to help@austnet.org should be done via mailing majordomo@austnet.org, with body of "subscribe help"
  • Public Forum Mailing List
    The User Mailing List is an open public mailing list available for AustNet clients. This mailing list is the main discussion forum for the management of the network, user requests and general discussion.

    • Individual help requests should be simply mailed to austnet@austnet.org
    • Permanent subscription to help@austnet.org should be done via mailing majordomo@austnet.org, with body of "subscribe austnet"
  • Services Management Mailing List
    The services mailing list is for the public access system related questions and discussion. This entails recommendations, enhancement notifications, channel and nickname registration disputes and/or mediation.

    • Individual help requests should be simply mailed to services@austnet.org
    • Permanent subscription to services@austnet.org should be done via mailing majordomo@austnet.org, with body of "subscribe services"
  • Network Topology and Routing Mailing List
    The Routing Mailing List contains discussion about AustNet's topology and server connectivity. This forum is responsible for server links, maintenance and announcements.

    • Individual help requests should be simply mailed to routing@austnet.org
    • Permanent subscription to routing@austnet.org should be done via mailing majordomo@austnet.org, with body of "subscribe routing"
