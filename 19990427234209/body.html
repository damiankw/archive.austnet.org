 Welcome!
Welcome to AustNet - a global network with servers in all corners of the globe. To learn more about how to start chatting on the AustNet Internet Relay Chat Network, read our New User's Guide. While you're here, why not explore services which AustNet makes available to you for free?

 Virtual World
AustNet has released Virtual World, which makes life harder for those who participate in nuking and other denial of service attacks by masking your real IP address. By doing this, they cannot resolve your IP address, and therefore prevents you from being nuked. Visit the Virtual World site for more info.

 Latest News
AustNet IRCD Patch - April 7, 1999
A post was recently sent to Bugtraq regarding a bug in AustNet's Virtual World code (which has been known for many months). A patch for austhex.servd7.4 has been released and can be downloaded here.

NOTE: This news item applies only to AustNet server administrators and other IRC networks using AustNet's IRCD.

portland.austnet.org delinked - March 29, 1999
In a recent vote by the AustNet staff, it was decided that portland.or.us.austnet.org was to be delinked from AustNet. Therefore we remind users that they should use an alternate US server. A complete list of AustNet servers can be found in our Server Catalogue.

AustNet Expands to Taiwan - March 16, 1999
We have approved a new server located in Taipei, Taiwan. This has also brough many new users to AustNet from Asia. Why not give it a try if you live in the region?
