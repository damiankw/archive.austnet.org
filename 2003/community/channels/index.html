Would you like your channel to be featured in our new "Featured Channel of the Week" page? If you would, send a 400-500 word profile in a plain text file to pr@austnet.org. A reporter will contact you if anything needs to be clarified. Your channel must fit the following to be featured:

More than 2 months old;
Friendly to new people;
Not have been suspended in the last six months;

Channels with a good reputation on Austnet will be given preference.

Some information you might like to include in your profile:

Channel name, owner's nickname, date created, recommended age group, type of channel (general chat, help, trivia etc), number of ops, number of bans, average number of users (not the peak!), URL if you have one, channel rules, topics usually discussed, channel history (has it changed owners? is there anything else interesting in the channel's history?), channel meets, if any, where the channel name comes from, funny things that have happened in the channel, quotable quotes from the channel and anything else interesting about the channel you would like to add.

Send us the file
The name of the file you send should be name.txt (eg: funkysexycool.txt) and you should spell-check the file before you send it in.

Optional Logo
If you would like to include a logo for your channel, it should be 280 pixels wide x 50 pixels high (maximum). It should be in .jpg or .gif format and be named with the name of your channel - eg: funkysexycool.gif. Attach to the same email as your text file profile.

Points to Note:
If you are not the owner, you must have the owner's permission to submit the information for Featured Channel of the Week. We will check this and any other information in the file before we go ahead and feature your channel.
