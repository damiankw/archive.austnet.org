This page is dedicated to the administration of Austnet and contains information including about the governing body, the network charter, votes which affect the network and its users, and information on servers and ASD.


AustNet is governed by an organisation charter. The charter provides both users and administrators with a central network ruling. To find out more, browse the AustNet Charter.


The AustNet Services Department (ASD) maintain services, and therefore we have strict controls on how issues are handled. To find out about these, visit the ASD Information site.


AustNet users agree to certain provisos when they connect to AustNet. These set forth in the User Agreement.


AustNet server applications are managed by the Server Application Centre. The fully automated centre allows you to view pending and past applications, submit server applications, and comment on a particular application.


Important decisions within AustNet are decided democratically by votes cast by the IRC operators and administrators of the network, and are collected by VoteOP. Visit the AustNet Voting Centre for an archive of current and past votes.
