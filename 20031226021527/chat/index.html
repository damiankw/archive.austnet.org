AustNet has provided a Java Chat module for you to use when you do not have access to an IRC client, or you may be blocked by a firewall. This module starts you off in one of 12 popular channels: #teens,#20+Swampboogie, #30&40's, #sydney, #perth, #canberra, #asianmelb, #macedonia, #jakarta, #beginner, #melbourne or #newbies. From there, you can use it as a normal IRC client to join other channels.
If you are going to be using IRC regularly, we suggest you download and learn to use an IRC client available from our software mirror.

To use the Java applet below, simply choose a channel to start in and type in a nickname to replace "AustUser" and click on Connect Now.

Once you have connected, you can join a new channel (not on the list) by typing /join #channelname
You can change your nickname after you have connected, by typing /nick yournickname eg: /nick sammyjnr
Open a private chat with another user by doubleclicking their nickname in the nickname list.

The Jpilot jIRC applet only supports Netscape 4.0, IE 4.0 or above.
Please grant permission when the pop up window appears from Netscape and IE.
