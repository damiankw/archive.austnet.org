Chatting

How do I send somebody a private message?

On just about any IRC client, use /msg nickname message. If that doesn't work use /privmsg nickname message.

How do I open an extended private chat?

Try /query nickname. On most clients this will activate a more convenient way to chat to a person, usually opening up a separate window.

What's DCC chat? How do I start one?

DCC means direct client-to-client. Once you set up a DCC chat messages pass straight from you to the recipient, bypassing the IRC network. To start one type /dcc chat nickname. Even if you become disconnected from IRC you can continue any DCC chats that you have open.

DCC chat doesn't open a window on my client. How do I use it?

Use the command /msg =nickname message (just like normal messaging with = prefixed to the nickname).
